# postoolspy
Global positioning solutions using combined GNSS and IMU.

## Supported Interfaces
- USB/UART
- NTRIP
- Mission Planner MAVLINK Server

## Supported Hardware

### RTK GNSS
1) u-blox 
    - ZED-F9P
    - NEO-M8P
2) Locosys
    - RTK1612-EVK
3) NMEA over USB

### IMU
1) None

### SLAM
1) None
